## Solutions & Commands
### OBS Video Source for Latency Testing
Recommended using a custom **HTML5 Browser Source** over a video loop for better accuracy and resource efficiency.

**Recommended HTML (`webrtc-test-source.html`)**:
- Displays real-time clock with milliseconds (`HH:MM:SS.mmm`).
- Includes a frame counter and a moving element (bouncing box) to visually identify frame drops or jitter.
- Use `requestAnimationFrame` for smooth 60fps rendering.

**Setup in OBS**:
1. Add Source > **Browser**.
2. Check "Local file" and select the `.html` file.
3. Set Resolution (e.g., 1280x720).
4. Click **Start Virtual Camera**.
